Photo Gallery Sample Application

This sample application is meant to be used with Getting Started Guide: AWS Web Application Hosting for Windows to demonstrate how to deploy an application using AWS.  This sample application demonstrates two approaches to uploading and displaying images for a website. The first approach, using Microsoft Access Database, is used so that you can quickly see your website deployed without making any necessary code changes. The second approach demonstrates how to make basic requests to Amazon Simple Storage Service (Amazon S3) using the AWS SDK for .NET and make basic requests to Amazon Relational Database Service (Amazon RDS) using MySQL Connector/Net 6.x. The sample application can be downloaded from https://s3.amazonaws.com/aws-sdk-samples/.NET/BaseImageGallery.zip.

Prerequisites

  • You must have a valid Amazon Web Services developer account.
  • Requires the AWS SDK for .NET, which is included in this package. For more information on the AWS SDK for .NET, see http://aws.amazon.com/sdkfornet.
  • Requires MySQL Connector/Net 6.x, which is included in this package. For more information on MySQL Connector/Net 6.x, see http://www.mysql.com/products/connector.
  • You must be signed up to use Amazon S3 and Amazon RDS. For more information on Amazon S3, see http://aws.amazon.com/s3. For more information on Amazon RDS, see http://aws.amazon.com/rds.

Running the Sample

To run this sample application, see the step-by-step instructions in the Deploy Sample Application section in the Getting Started Guide: AWS Web Application Hosting for Windows Guide.
